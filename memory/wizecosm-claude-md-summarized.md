**WizeCosium Project Documentation Summary:**

After consolidating all folder summaries into root `CLAUDE.md`, the following structure was established:

### Folder Summaries Added to CLAUDE.md:

1. **00 - Univers** (processed separately via init)
   - World-building lore, magic system (Cosm), factions, races origins
   
2. **01 - Géographie** 
   - Countries, capitals, continents geographic database organized by region type with 3 main subdirectories (Capitales, Continents, Pays)

3. **02 - TimeLine
   - Story chronology via mermaid diagram showing phase structure across eras and events like "La pluie d'étoiles"

4. **03 - Personnages** 
   - ~19 character files including main cast (Hank, Jeanne, Öko, Taneï) plus Plénitude faction members
   - Contains template file for new characters and per-character CLAUDE summaries

5. **04 - Arcs** 
   - Complete plot structure across 6 major phases from preface through final arc finale
   - Each phase includes scene breakdowns (P1S1-P1S7) and narrative files

### Key Story Elements Identified:
- Magic system called "Cosm" distributed unequally between peoples  
- Yamés vs Gamé faction conflict spanning generations
- Core protagonists: Hank, Jeanne, Öko, Taneï forming central trio
- Antagonist forces led by Plénitude under Paléisto's manipulation
- Story timeline begins after "La Guerre des Cosmers" with 15 year epilogue gap

**CLAUDE.md now serves as master navigation file at project root.**


Why: To preserve the consolidated folder structure summary for future reference and quick project orientation without re-processing each subfolder.
