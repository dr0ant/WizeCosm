

## 01 - Géographie
**Purpose:** Geographic locations including countries, capitals, continents, and regional world information for WizeCosm setting.

**Structure:** Main folder containing geographic data organized by region type:

- Capitales (capital cities database) 
- Continents (major landmasses overview)
- Pays (nations/regional details with 10+ subfolders each)


## 02 - TimeLine  
**Purpose:** Complete story chronology and narrative timeline via mermaid diagram showing phase structure.

**Structure:** Organized by story beats:

- Les Eres (eras documentation + multiple Home files)
- Phase-based progression from backstory through final arc
15 years time gap markers, key events like "La pluie d'étoiles"


## 03 - Personnages 
**Purpose:** Character profiles with backstories, personalities, relationships and development arcs for main cast including Plénitude faction members (Paléisto, Safiya et al.).

**Structure:** Extensive character database:

- ~19 main character files (Hank, Jeanne, Öko, Taneï, Bhaus etc.)
- Personnages - Plénitude subfolder for antagonist characters
- Template file and CLAUDE summary docs maintained per character


## 04 - Arcs 
**Purpose:** Narrative story phases from preface through final arc — complete plot structure spanning 12+ numbered beats across multiple arcs.

**Structure:** Organized by production phase:

- 00 - Preface (pre-story prologue material)
- Phase 1/2/3/4 each with scene breakdowns and narration files  
- Arc 1 continuation covering journey events, tournament arc finale
